## 💼 Sobre o Projeto

> 💡 Um sistema desenvolvido com base em um **problema real da área da saúde**

O **Prontuário Psicológico Digital** é uma aplicação front-end criada para o serviço de Atendimento-Escola da **Faculdade de Ciências Médicas da Santa Casa de São Paulo**.

Nesta fase, o foco está na construção da **interface e experiência do usuário**, simulando o funcionamento de um sistema clínico completo.
